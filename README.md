# ✨ Features

### 🎥 Playback

- Hardware Accelerated Playback
- Smooth 4K Playback
- Playback Speed Controls
- Resume Playback
- Background Playback
- Multiple Aspect Ratios

### 👆 Gesture Controls

- Double Tap to Seek
- Brightness Control
- Volume Control
- Swipe to Seek
- Screen Lock

### 💬 Subtitle Support

- Online Subtitle Download
- External Subtitle Files
- Subtitle Styling
- Multiple Subtitle Formats

### 📂 Library

- Automatic Video Scan
- Folder Browser
- Recent Videos
- Video Information
- HD & UHD Detection

### 🎨 Design

- Material Design 3
- Premium Animations
- Beautiful Typography
- Dark Theme
- Responsive UI
