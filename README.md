## ✨ Overview

CoverPlay is a premium Android video player designed with a clean interface, smooth animations, and powerful playback controls. It combines beautiful design with advanced functionality to provide an exceptional local media experience.

---

## 🚀 Features

### 🎥 Playback
- Ultra-smooth video playback
- Hardware acceleration
- Resume playback
- Background playback support
- Multiple aspect ratios
- Playback speed controls

### 👆 Smart Gestures
- Double tap to seek
- Swipe to adjust brightness
- Swipe to adjust volume
- Swipe to seek
- Screen lock mode

### 💬 Subtitles
- Automatic online subtitle search
- External subtitle support
- Subtitle customization

### 📁 Library
- Scan local videos
- Folder browsing
- Recent videos
- Video information
- HD/4K video support

### 🎨 Design
- Modern Material Design 3 UI
- Beautiful typography
- Smooth animations
- Premium interface
- Dark theme
